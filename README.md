# Regime Aware Quantitative Trading System

This project implements a regime aware quantitative trading system for NIFTY derivatives using Python.

## Author
Priyanshu Maheshwari  
Roll No.: 24/SCA/MCA/038  

## Project Overview
The system identifies market regimes using statistical models and applies a trading strategy only during favorable conditions.

## Key Components
- Feature engineering using price and options data
- Regime detection using Hidden Markov Model
- EMA based trading strategy
- Machine learning based trade filtering
- Outlier trade analysis

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- XGBoost

## Files
- Project Report (Word document)
- Jupyter Notebook with complete code

## Note
Synthetic futures and options data are used due to unavailability of stable intraday NSE derivative APIs.
